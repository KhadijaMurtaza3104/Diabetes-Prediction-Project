# Diabetes Prediction Project

## Description

This project predicts whether a patient has diabetes using a Logistic Regression model.

## Dataset

Pima Indians Diabetes Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- FastAPI
- Uvicorn

## Features

- Data Cleaning
- Exploratory Data Analysis
- Logistic Regression Model
- FastAPI Deployment

## Run the Project

```bash
python -m uvicorn app:app --reload
```

Open:

```
http://127.0.0.1:8000/docs
```

to test the API.
